# 🎴 타로 카드 리딩 애플리케이션

PyQt5 기반 타로 카드 뽑기 GUI 애플리케이션

## ✨ 주요 기능

- **5가지 스프레드 지원**
  - 원 카드 (1장): 오늘의 운세, 간단한 질문
  - 쓰리 카드 (3장): 과거-현재-미래
  - 관계 스프레드 (7장): 나와 상대방의 관계 분석
  - 켈틱 크로스 (10장): 가장 포괄적인 종합 리딩
  - 커스텀 (1~10장): 자유 선택

- **화려한 UI/UX**
  - 클래식한 타로 카드 뒷면 디자인
  - 페이드인 애니메이션 효과
  - 다크 테마
  - 가로 스크롤 지원 (많은 카드)

- **편의 기능**
  - 리딩 히스토리 저장/조회
  - 전체화면 모드 (F11 또는 버튼)
  - 음소거 기능
  - 중복 방지 (한 세션에서 같은 카드 두 번 안 나옴)

- **🔮 AI 해석 기능** (데스크톱 앱 전용)
  - Llama 3.1 기반 AI 타로 카드 해석
  - 스프레드와 카드 위치를 고려한 심층 분석
  - 카드 조합에 따른 종합적인 메시지 제공
  - 실용적이고 긍정적인 조언
  - 완전 로컬 실행 (인터넷 불필요)

## 📁 프로젝트 구조

```
Taro/
├── cards/              # 타로 카드 이미지 (78장)
├── assets/
│   ├── card_back.png   # 카드 뒷면 이미지
│   └── sounds/         # 사운드 파일 (선택사항)
├── data/
│   └── history.json    # 리딩 히스토리
├── tarot_app.py        # 메인 애플리케이션
├── spreads.py          # 스프레드 정의
├── card_manager.py     # 카드 관리
├── ai_interpreter.py   # AI 해석 모듈
├── create_card_back.py # 카드 뒷면 생성 스크립트
├── test_ai.py          # AI 기능 테스트
├── run.bat             # Windows 실행 스크립트
└── requirements.txt    # 필요한 패키지
```

## 🚀 설치 및 실행

### 1. 필수 패키지 설치

```bash
pip install -r requirements.txt
```

필요한 패키지:
- PyQt5 (GUI)
- Pillow (이미지 처리)
- pygame (사운드, 선택사항)

### 2. AI 해석 기능 설정 (선택사항)

AI 해석 기능을 사용하려면 Ollama와 Llama 3.1 모델이 필요합니다.

```bash
# Ollama 설치 (https://ollama.com)
# Windows: 설치 프로그램 다운로드

# Llama 3.1 모델 다운로드
ollama pull llama3.1:8b

# AI 기능 테스트
python test_ai.py
```

### 3. 앱 실행

**Windows:**
```bash
run.bat
```

**Linux/Mac:**
```bash
python tarot_app.py
```

## 🎮 사용 방법

### 메인 메뉴
1. 원하는 스프레드 버튼 클릭
2. 커스텀의 경우 카드 수 선택

### 카드 뽑기
1. "카드 뽑기" 버튼 클릭
2. 왼쪽부터 순서대로 카드가 공개됨
3. 각 카드 아래에 위치 의미 표시
4. 최소 1장 이상 뽑으면 "AI 해설" 버튼 활성화
5. "AI 해설" 클릭 시 AI가 카드를 분석하여 해석 제공
6. 모든 카드를 뽑으면 "저장" 가능

### 히스토리
- "히스토리 보기"에서 과거 리딩 확인
- 항목 더블클릭으로 상세 보기
- "전체 삭제"로 히스토리 초기화

### 단축키
- **F11**: 전체화면 토글
- **ESC**: 전체화면 종료

## 🐛 문제 해결

### PyQt5 설치 실패
```bash
# Windows
pip install PyQt5-sip PyQt5

# 또는 conda 사용
conda install pyqt
```

### 카드 이미지를 찾을 수 없음
- `cards` 폴더가 `tarot_app.py`와 같은 위치에 있는지 확인
- 카드 이미지가 `.jpg` 확장자인지 확인

### 애니메이션이 작동하지 않음
- PyQt5 버전 확인 (5.15.0 이상 권장)
